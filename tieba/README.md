# 百度贴吧签到脚本

用于青龙面板的百度贴吧自动签到脚本。

## 环境变量

| 变量名 | 说明 |
|--------|------|
| `BD_COOKIE` | 百度系通用 Cookie（贴吧+网盘共用） |

## 获取 Cookie 方法

### 方法一：浏览器开发者工具（推荐）

1. 打开百度贴吧网站 https://tieba.baidu.com
2. 登录你的百度账号
3. 按 `F12` 打开开发者工具
4. 切换到 `Network`（网络）标签
5. 刷新页面，点击任意一个请求
6. 在请求 Headers 中找到 `Cookie` 项
7. 复制完整的 Cookie 字符串

### Cookie 格式

脚本支持完整的 Cookie 字符串：
```
BDUSS=你的BDUSS值; STOKEN=你的STOKEN值; BAIDUID=你的BAIDUID值; ...
```

## 在青龙面板配置

1. 进入青龙面板 → 环境变量
2. 新建变量：
   - 名称：`BD_COOKIE`
   - 值：粘贴你的百度 Cookie
3. 保存

## 定时任务

建议添加定时任务，每天自动执行：

- Cron 表达式：`0 8 * * *`（每天早上 8 点）

## 注意事项

- 贴吧和网盘签到可以共用同一个 `BD_COOKIE`
- BDUSS Cookie 有效期较长，但建议定期更新
- 同一个账号多个贴吧签到可能需要较长时间
