# 青龙面板配置说明

## 📋 基本信息

- **脚本文件**: `iqiyi_ql.js`
- **变量名**: `IQIYI_COOKIE`
- **推荐 Cron**: `0 8 * * *` (每天 8:00 执行)

---

## 🔧 添加脚本

### 方式一：订阅管理 (推荐)

1. 打开青龙面板 → **订阅管理** → **新建订阅**
2. 填写信息:
   - **名称**: `爱奇艺签到`
   - **类型**: `公开仓库` 或 `私有仓库`
   - **链接**: 你的脚本仓库地址
   - **定时规则**: `0 8 * * *`
3. 保存并运行

### 方式二：手动添加

1. 打开青龙面板 → **脚本管理**
2. 点击 **添加脚本**
3. 填写信息:
   - **文件名**: `iqiyi_ql.js`
   - **内容**: 粘贴脚本代码
4. 保存

---

## 🍪 配置环境变量

### 单账号配置

1. 打开青龙面板 → **环境变量**
2. 点击 **添加变量**
3. 填写:
   - **名称**: `IQIYI_COOKIE`
   - **值**: `P00001=xxx; P00002=xxx; P00003=xxx; __dfp=xxx; QC005=xxx;`
   - **备注**: `爱奇艺Cookie - 账号1`

### 多账号配置

**方式一：换行分隔 (推荐)**

```
P00001=账号1的值; P00002=账号1的值; P00003=账号1的值;
P00001=账号2的值; P00002=账号2的值; P00003=账号2的值;
P00001=账号3的值; P00002=账号3的值; P00003=账号3的值;
```

**方式二：多变量名**

| 变量名 | 值 |
|--------|-----|
| `IQIYI_COOKIE` | `P00001=账号1; ...` |
| `IQIYI_COOKIE_1` | `P00001=账号1; ...` |
| `IQIYI_COOKIE_2` | `P00001=账号2; ...` |
| `IQIYI_COOKIE_3` | `P00001=账号3; ...` |

## ⏰ 设置定时任务

### 方式一：通过脚本文件头

脚本已内置:
```javascript
// cron: 0 8 * * *
// new Env('爱奇艺签到');
```

青龙面板会自动识别并创建定时任务。

### 方式二：手动添加定时规则

1. 打开 **定时任务**
2. 找到 `iqiyi_ql.js` 脚本
3. 编辑 → 修改 **定时规则**
4. 常用规则:
   - `0 8 * * *` - 每天 8:00
   - `0 9,18 * * *` - 每天 9:00 和 18:00
   - `30 7 * * *` - 每天 7:30

## 🔍 如何获取 Cookie

1. 打开 [爱奇艺官网](https://www.iqiyi.com) 并登录
2. 按 `F12` 打开开发者工具
3. **Application** → **Cookies** → `https://www.iqiyi.com`
4. 复制以下字段值:
   - `P00001` ✅ 必需
   - `P00002` ✅ 必需
   - `P00003` ✅ 必需
   - `__dfp` (可选)
   - `QC005` (可选)

## ⚠️ 常见问题

### Q: 提示 "未设置环境变量 IQIYI_COOKIE"

检查环境变量是否正确添加并**启用**。

### Q: Cookie 失效

爱奇艺 Cookie 有效期约 30 天，需定期更新。

### Q: 多账号只签到了一个

确保使用换行分隔或 `IQIYI_COOKIE_1, IQIYI_COOKIE_2` 格式。

### Q: 如何测试脚本

青龙面板 → **脚本管理** → 找到脚本 → **运行**

---

## 📌 注意事项

1. **Cookie 安全**: 不要泄露或上传到公开仓库
2. **定期更新**: Cookie 过期后需重新获取
3. **频率限制**: 脚本已内置请求间隔，无需担心
4. **活动时效**: 部分抽奖活动可能已下线
