# 爱奇艺签到脚本

自动完成爱奇艺每日签到任务，获取成长值和抽奖奖励。

## 功能

- ✅ 查询账号信息和 VIP 状态
- ✅ 每日签到获取成长值
- ✅ 每天摇一摇抽奖
- ✅ 白金会员抽奖
- ✅ V7 免费升级星钻

## 环境变量

| 变量名 | 必填 | 说明 |
|--------|------|------|
| `IQIYI_COOKIE` | 是 | 爱奇艺完整 Cookie |

## Cookie 获取方法

1. 浏览器打开 [爱奇艺官网](https://www.iqiyi.com) 并登录
2. 按 `F12` 打开开发者工具
3. 切换到 `Network` 标签
4. 刷新页面，点击任意请求
5. 在 `Headers` 中找到 `Cookie` 字段，复制完整内容

## 青龙面板配置

1. 环境变量 → 添加变量
   - 名称：`IQIYI_COOKIE`
   - 值：复制的 Cookie 内容

2. 定时任务 → 添加任务
   - 名称：`爱奇艺签到`
   - 命令：`task iqiyi/iqiyi_checkin.py`
   - 定时：`5 0 * * *`

## 定时建议

建议每天凌晨执行：`5 0 * * *`（每天 00:05）

## 注意事项

- Cookie 有效期较长，但建议定期检查更新
- 非 VIP 用户部分功能可能无法使用
- 如遇问题请重新获取 Cookie
